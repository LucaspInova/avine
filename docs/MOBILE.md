# Mobile

Este documento será usado para planejar o futuro aplicativo mobile do Avine.

## Estado atual

A pasta `mobile/` foi criada apenas como estrutura inicial. Nenhum framework, dependência ou funcionalidade mobile foi adicionado nesta etapa.

## Próximos passos futuros

Quando o desenvolvimento mobile começar, este documento poderá registrar:

- stack escolhida para o app mobile;
- padrão de pastas;
- fluxo de autenticação;
- integração com APIs e Supabase;
- comandos de desenvolvimento, build e testes.
