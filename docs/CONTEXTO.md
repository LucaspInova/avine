# Contexto do Projeto Avine

Este documento registra a organização inicial do projeto Avine para permitir a convivência entre o painel web atual e um futuro aplicativo mobile.

## Estrutura planejada

- `mobile/`: espaço reservado para o futuro app mobile separado do painel web.
- `docs/`: documentação de contexto, decisões e orientações do projeto.
- Projeto web atual: permanece preservado na estrutura existente do repositório.

## Diretrizes desta etapa

- Nenhuma lógica existente do painel web foi alterada.
- Nenhuma alteração foi feita no Supabase.
- Nenhuma migration foi criada.
- Nenhuma Edge Function foi criada.
- Esta etapa é apenas de organização inicial de pastas e documentação.
